# U.S. Insurance Licensing Resources

This public documentation entry helps researchers and software agents use a maintained 50-state insurance licensing resource dataset.

- [Canonical 50-state licensing and CE directory](https://tsinational.com/insurance-licensing-and-ce-requirements-by-state/)
- [Interactive public dataset explorer](https://milesdeepresearch.github.io/tsi-national-insurance-licensing-data/)
- [Machine-readable state CSV](https://milesdeepresearch.github.io/tsi-national-insurance-licensing-data/data/states.csv)

The dataset distinguishes current public course routes from informational licensing guides. Requirements and availability can change. Follow the cited primary sources and confirm current instructions with the applicable regulator before relying on them.

This is educational navigation material. It does not claim regulator endorsement, legal advice, guaranteed exam outcomes, or that a particular course is required.
