U.S. Insurance Licensing Resources - 50-State Dataset
=======================================================

Machine-readable and human-readable links for insurance licensing research across all 50 U.S. states. This resource is maintained by **TSI National (Insurance Training Professionals LLC)**.

- `Interactive 50-state explorer <https://milesdeepresearch.github.io/tsi-national-insurance-licensing-data/>`__
- `Insurance licensing and CE requirements by state <https://tsinational.com/insurance-licensing-and-ce-requirements-by-state/>`__
- `Download the current canonical CSV <https://tsinational.com/insurance-licensing-and-ce-requirements-by-state.csv>`__
- Repository data files: `CSV <data/states.csv>`__ and `JSON <data/states.json>`__

The directory distinguishes currently available TSI course routes from informational licensing guides. Availability can change, so the canonical TSI directory is the current source.

This is an educational navigation resource. It does not claim state approval, endorsement, legal advice, exam guarantees, or that a course is required.
